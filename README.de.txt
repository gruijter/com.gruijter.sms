Eine kleine Homey-App zum Senden von SMS-Nachrichten über mehr als 80 SMS-Anbieter.

Nachdem die Einstellungen für den SMS-Anbieter vorgenommen wurden, den Sie verwenden möchten, können Sie eine SMS über einen Fluss senden.

* 46elks
* AspSMS
* BulkSMS
* Cheapconnect.net (nl)
* Clickatell
* CM Direct
* Free Mobile (fr)
* Gatewayapi
* Messagebird
* SendInBlue
* Sms77
* TargetSMS
* Twilio
* Spryng
* Textbelt
* DellMont: Dies umfasst mehr als 60 Voip-Anbieter wie Voipbuster, Freecall, Cheapvoip usw.
